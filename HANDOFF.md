## 🎯 作品の核（Bibleからの抜粋）

### コンセプト
**「三層詐欺アニメ」** — SNSの顔 × 学校の顔 × Discord越しの素の顔。  
4人とも3つの仮面を持ってる。「昼と夜」じゃなく「三層」。

### 共通モチーフ
> **「名前通りじゃない奴しか、いらない」**

4人全員が時間帯の名前（朝目・真昼・日暮れ・夜虎）：
- あさめ（朝）×朝苦手 ＝ 名前詐欺
- まひる（昼）×昼間病み ＝ 名前詐欺
- ひぐれ（夕）= 夕方絵描いてる、名前通り
- やとら（夜）= 夜起きてる、名前通り

→ **名前通りじゃない者×名前通り者で、夜にだけ揃う**

### 看板ギミック
1. ラスト `SYSTEM > ○○ joined the game` ＋ モノローグ締め
2. Cスタ冒頭に**あさめのTikTok数字**（世間温度計）
3. **名前詐欺チャート**を毎話更新

### シーズン貫通伏線
1. **やとら5年孤独の真相**（最終回回収）
2. **まひる病み垢の最初の投稿**（中盤回収）
3. **ひぐれのクリーパーキーホルダー**（終盤回収＝幼馴染の片想いフラグ）

## 🔥 バズtracker仕様（重要）

### アーキテクチャ
```
GitHub Pages → 起動時に raw.githubusercontent.com から
              data/weekly-shorts.json 読込（認証不要）
            ↓
         localStorage キャッシュ
            ↓
       ユーザー編集
            ↓
   💾 GitHubに保存ボタン
            ↓
  GitHub Contents API (PAT必要)
            ↓
   data/weekly-shorts.json コミット
```

### API取得バックエンド（フェイルオーバー）
1. Piped API（7インスタンス）
2. Invidious API（6インスタンス）
3. CORS Proxy + Invidious

### デバッグログ（左下📋ボタン）
全API呼び出しのHTTPステータス、経過時間、エラーが見れる。  
「📋 全文コピー」でログ取得可能。

### Token保存
- localStorage の `gh-pat-weekly-shorts`
- GitHubに送信されない（認証ヘッダーで使うだけ）
