🔥 トレンド動画を自動取得

Piped API（YouTube公開API、APIキー不要）経由でバズってるマイクラ動画を一括取得。プリセットを選んでください。重複は自動スキップ。

※ Piped → Invidious → CORS Proxy の順で多重フェイルオーバー。
※ 取得失敗時はログパネル（左下📋ボタン）で詳細確認。各APIインスタンスのHTTPステータス・経過時間が見れる。
※ 重複動画は自動スキップ。地域・カテゴリは自動推定。
※ ヘルスチェックで事前にどのインスタンスが生きてるか確認できます。

📡 監視チャンネル

登録したチャンネルのRSSフィードから最新動画を一括取得（YouTube公式・絶対落ちない）。
📥 全チャンネル更新で全部の最新15本を取得→重複自動スキップ。

📚 推奨チャンネルから一括追加（クリックで展開）

※ RSS = YouTubeの公式フィード。Piped/Invidiousと違って絶対に落ちない。
※ 取得経路：CORS Proxy → YouTube RSS（直接アクセスはCORSで失敗するためProxy必須）。
※ 1チャンネルあたり最新15本まで。それ以上はYouTubeの仕様で取得不可。

📋 URL一括追加

YouTube URLを改行区切りで何個でも貼り付け。タイトル・チャンネル名はnoembed APIで自動取得。

🔑 YouTube Data API キー管理

複数キーローテートで quota分散。1キー10,000units/日、3キーで30,000units/日。
trackerは YouTube API → Piped → Invidious → CORS Proxy → RSS の優先順で取得します。

⚠️ セキュリティ重要：
1. このリポジトリはpublic。コードに直書きすると GitHub Secret Scanning が検出 → Google が即無効化。
2. キーはこのフォーム経由でlocalStorage保存のみ（GitHubに送信されない）。
3. 必ず Cloud Console で各キーに HTTP referrer制限 を設定（許可: https://extraoverone.github.io/* + file:///*）。
4. API制限 も「YouTube Data API v3 のみ」に絞る（他APIへの悪用防止）。

📚 APIキー取得手順（クリックで展開）
  1. Google Cloud Console にログイン
  2. 上部「プロジェクト選択」→ 新しいプロジェクト（名前は何でもOK、例: yofukashi-1）
