01 — MECHANICS POOL使える固有メカニクス

地下都市とウォーデンだけが持つ仕様。これを笑いに変換する。

盲目ウォーデンは目が見えない。動かなければ気付かれない。
振動探知音・足音・通知音・心拍…全部バレる。声を出したら即死。
嗅覚見えなくても匂いで追跡。香水・汗・食べ物バレる。
ソニックブーム壁貫通の遠距離攻撃。隠れても逃げられない。
4回シュリーカー3回目の警告音で止めればギリ召喚されない。チキンレース成立。
羊毛ブロック唯一振動を伝えない救世主。絨毯作戦が可能。
Swift Sneak地下都市限定エンチャ。無音移動ができる。宝探しネタ。
回収コンパス死んだ場所を指す。地下都市で死ぬ→またそこへ導かれる無限地獄。
壊れたポータル未実装の謎のポータル。起動したら何処かヤバい次元へ（創作自由）。
リインフォースド・ディープスレート絶対に壊れない壁。絶望の象徴。

03 — CONCEPT POOL未開発パターンプール

まだフル台本化していないアイデア。今後、深淵/ウォーデン系の話数を書く時の種。

PATTERN A

ウォーデン育児中

古代都市に侵入したら、ウォーデンが子ウォーデンを寝かしつけ中。新ルール「赤ちゃん起こしたら即死」。ウォーデン＝母性系。子ウォーデンが泣き出し、親ウォーデンが怖い顔で4人を睨む→「失礼します」と静かに退場。
核：怖いもの＝実は日常のギャップ萌え。ビジュアル強い。
PATTERN C

サイレント・ゲームショー

深淵潜入をゲームショー形式で実況。ノイズメーター装備。「まひる、ため息！-10点！」。各キャラの「音を出す理由」でキャラ紹介：あさめ陽キャ、ひぐれ運動部反応、やとらクール発言、まひる無言。最後まで生き残るのはまひる（病み、一番音を出さない）。
核：フォーマットの新規性。キャラ性の可視化。
PATTERN D

やとらの道間違え

張り詰めた10秒、ウォーデン出現！→ ただのスケルトン。ここ古代都市じゃなくて普通の洞窟。やとらの指揮と講義、全部無駄。「……あれ」のクール崩壊。
核：クール崩壊ベタ笑い。読めるけど短尺なら成立。
PATTERN E

ひぐれ運動部無双

ウォーデンのソニックブームを、ひぐれがバスケのクロスオーバーで回避。サッカー部の走力で逃げ切る。「部活の応用でボス撃破」。運動神経とマイクラの異次元融合。
核：ひぐれの新設定（運動部すけっと）フル活用。爽快感。
PATTERN F

回収コンパス無限ループ

深淵で4人死亡 → リスポーン → 回収コンパスが深淵の入口を指す → 無意識に歩いて戻る → また死ぬ → コンパスが指す → また戻る。ウォーデンが「また来たな」の顔。行動の無限ループ化。
核：マイクラ固有アイテム（回収コンパス）の皮肉な使い方。
PATTERN G

壊れたポータル起動

古代都市の「壊れた未実装ポータル」を4人が修理起動したら、想定外の次元へ。オチの次元は自由設定 — 例：「Among Us世界」「バグった学校」「現実の視聴者の部屋」など、創作自由。
核：謎のロアを利用、ネザー/エンドじゃない第3の飛び先。
PATTERN H

Swift Sneak争奪戦

地下都市限定エンチャント「Swift Sneak」を巡って4人が泥棒合戦。静かに動く必要があるのに、奪い合いで振動発生→ウォーデン召喚。手に入れた奴は学校の廊下も無音で移動できる特権を得る。
核：レア報酬を取り合うチャレンジ型。今後の日常回につながる。
PATTERN I

盲目ウォーデンと目合わせ

4人がウォーデンの真ん前に立って動かない。「盲目だから見えない」。絶対的安全。でも全員、別々の理由で動いちゃう（あさめ：TikTok撮りたい、ひぐれ：くしゃみ、まひる：スマホ通知）。一人ずつ脱落していく脱出ゲーム。
核：盲目メカニクスをキャラ毎の失敗で見せる。
