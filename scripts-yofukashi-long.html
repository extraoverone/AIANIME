よふかしくらぶ！横動画 台本集

YouTube長尺（8〜20分）。ショートで引き込んだ視聴者が「この4人をもっと見たい」と思って来る場所。全40案。

横動画の設計思想

  • ショートとの役割分担：ショート＝認知（「この4人面白い」）、横動画＝定着（「この4人をずっと見ていたい」）
  • 横動画でしかできないこと：キャラの掘り下げ、長い掛け合い、伏線回収、空気感、ゆるい雑談
  • 最初の30秒で勝負：横動画でも冒頭フックは必須。「今日のよふかしくらぶ、いつもと違う」と思わせる
  • ミッドロール誘導：8分以上でミッドロール広告が入る。8分は最低ライン。

マイクラ直球系 8本

マイクラのゲーム体験そのもので勝負する。プレイ映像＋掛け合いが核。

Y01「4人で100日サバイバルしたら全員の性格が出すぎた」

マイクラチャレンジ15-20分
100日サバイバルのダイジェスト。Day 1→10→30→50→100と飛ばしながら、各キャラの拠点・プレイスタイルの変化を追う。あさめの家が毎回全壊してるのに毎回「新しい家建てた！」と元気なのがオチ。
構成：Day1 スポーン混乱 → Day10 各自の拠点紹介 → Day30 初ネザー遠征 → Day50 共同建築開始 → Day70 エンダーアイ集め → Day100 エンドラ討伐 → エンディング
伸びる理由：「100日」は検索ワードとして強い。ダイジェスト形式でテンポよく見せつつ、キャラの成長が感じられる。

Y02「マイクラの全バイオーム制覇の旅」

マイクラ12-15分
全バイオームを巡る旅。バイオームを見つけるたびにリアクションチェック。各バイオームで1つ「やること」を決めてから次へ。砂漠で砂城、ジャングルでターザン、雪原で雪合戦……。やとらだけ各バイオームの豆知識を語る。
構成：出発宣言 → 草原→砂漠→ジャングル→雪原→沼地→メサ→キノコ島 → 各バイオームで1ミッション → 最後のレアバイオーム発見で全員叫ぶ
伸びる理由：旅系は視聴維持率が高い。「次はどこ？」で見続ける。各バイオームがパターンインタラプトになる。

Y03「マイクラの最強装備を全員分揃えるまで帰れない」

マイクラチャレンジ15分
ネザライトフル装備×4人分＝16セット。分業するが効率が最悪。あさめがダイヤを溶岩に落とす。ひぐれが古代の残骸の掘り方を知らない。まひるがネザー要塞で迷子。やとらだけ黙々と効率的に進める。
構成：目標宣言 → 役割分担（破綻） → ダイヤ採掘編 → ネザー編 → 古代の残骸地獄 → エンチャント → 完成＆記念撮影
伸びる理由：「帰れない」系は鉄板。進捗バーを常に画面端に出すことで「あとどれくらい？」が気になる。

Y04「やとらのレッドストーン講座（を3人が全力で妨害する）」

マイクラコメディ教育10-12分
やとらが真面目にレッドストーン回路を教えようとするが、あさめがスイッチを勝手に押す、ひぐれが回路の上を歩く、まひるがTNTを接続する。教育動画の皮をかぶった妨害コメディ。でも最後にちゃんと動く装置が完成する。
構成：「今日は自動農場を作ります」 → 基礎説明（妨害1） → 回路組み立て（妨害2） → テスト運転（大爆発） → やり直し → 奇跡の完成
伸びる理由：教育系＋コメディのハイブリッド。レッドストーンの仕組みが自然と学べるので保存率が高い。「自分もやってみたい」コメント。

Y05「初見でネザー行ったら全員パニックになった」

マイクラコメディ12分
ひぐれの初ネザー。ポータルをくぐった瞬間のリアクション。ガストに泣く。マグマキューブに困惑。ブレイズに全滅。ネザー要塞で迷子。「もう二度と来たくない」と言いながら帰り道がわからない。
構成：ポータル作成（緊張） → 初ネザー入場（衝撃） → 探索（恐怖） → ガスト遭遇（パニック） → 要塞発見 → ブレイズ戦（全滅） → 帰還……できない → 奇跡の帰還
伸びる理由：初ネザーの感動はマイクラプレイヤー全員の原体験。ひぐれの素朴なリアクションが視聴者の「あの頃」を思い出させる。

Y06「4人のワールド、1年後どうなった？」

マイクラエモ10分
サーバー開始から1年。ワールドを上空から俯瞰。最初の豆腐ハウスがまだ残ってる。各拠点を巡りながら「ここでこんなことあったよね」と振り返る。最後に4人で新しい建築を始めて「まだまだ続くよ」。
構成：空撮で全体像 → 最初のスポーン地点 → 各拠点の歴史 → 名場面の場所巡り → 「ここ覚えてる？」 → 新しい場所に新拠点 → 未来へ
伸びる理由：シリーズ追ってる人への最強ファンサ。新規には「こんなに歴史あるのか、最初から見たい」と過去動画への導線。

Y07「マイクラのアプデ全部試す会」

マイクラ教育コメディ12分
最新アプデ（Tiny Takeover / Chaos Cubed）の新要素を4人で全部検証する。金のタンポポ実験、サルファーキューブの物理テスト、新バイオーム探検。やとらが解説担当だが、あさめが勝手に触って毎回カオスになる。
構成：アプデ概要 → 新アイテム検証×3 → 新MOB検証 → 新バイオーム探検 → お気に入り投票 → 次のアプデ予想
伸びる理由：アプデ直後の検索需要＋教育系の保存率。定期フォーマットとして毎回のアプデでシリーズ化。

Y08「深夜のマイクラ釣り大会 — 一番レアなもの釣った人が勝ち」

マイクラチャレンジ10分
夜の海辺で4人が釣り竿を垂らす。ゆるい雑談をしながら釣果を競う。魚が釣れるたびにリアクション。エンチャント本が出たら大騒ぎ。まひるだけ「ゴミ」を引く。最後にまひるが名無しの地図を釣って宝の地図だと判明。
構成：海辺集合 → ルール説明 → 釣り開始（雑談タイム） → レア釣果で盛り上がり → まひるの連続ゴミ → 逆転の宝の地図 → 宝探しへ（次回予告）
伸びる理由：ゆるい空気感が横動画の強み。雑談が面白ければ内容は何でもいい。釣りの「何が出るか」はガチャと同じドーパミン構造。

チャレンジ・対決系 6本

ルールを決めて競う。勝敗がつくので最後まで見る理由が生まれる。

Y09「マイクラかくれんぼ — 隠れる側はブロックに擬態していい」

チャレンジコメディ12分
鬼1人、隠れる側3人。隠れる側はブロックを1つ選んで自分の周りに同じブロックを積んで擬態できる。制限時間5分で見つからなければ勝ち。あさめが花畑に花として隠れる。まひるが黒曜石の壁に溶け込む。ひぐれが丸石の中に隠れるが明らかにはみ出てる。
構成：ルール説明 → Round 1（やとら鬼） → Round 2（あさめ鬼） → Round 3（まひる鬼） → 最終Round → MVP発表
伸びる理由：かくれんぼは全世界共通の遊び。擬態ルールがマイクラならではの独自性。視聴者が「そこにいる！」と叫びたくなる。

Y10「嘘つきマイクラ — 1人だけ違う世界にいる」

チャレンジコメディ12分
4人のうち1人だけ別のシード値のワールドにいる。質問し合って「誰が嘘つき（別ワールド）か」を当てる。「スポーン地点の近くに村ある？」「海見える？」——嘘つきは他の3人の会話に合わせてテキトーに答える。
構成：ルール説明 → 質問タイム（3分） → 推理タイム → 投票 → 正解発表 → 2戦目（嘘つき交代） → 3戦目
伸びる理由：人狼のマイクラ版。「誰が嘘つき？」は最後まで見る動機として最強。キャラの嘘のつき方に性格が出る。

Y11「30分でお城を建てろ — 4人同時建築バトル」

チャレンジ建築15分
制限時間30分で「城」をテーマに建築。完成後に互いの城を見学して投票。あさめ＝カラフルだがぐちゃぐちゃ。まひる＝不気味だが独創的。ひぐれ＝四角い。やとら＝完璧だが時間切れで片面だけ。
構成：テーマ発表 → 制作タイム（タイムラプス＋実況交互） → 完成 → 見学ツアー → 投票＆講評 → 敗者罰ゲーム
伸びる理由：建築系は安定して人気。タイムラプスが映像として美しい。完成品のギャップが笑いになる。

Y12「マイクラ脱出ゲーム — やとらが作った牢獄から脱出しろ」

チャレンジマイクラ15分
やとらがレッドストーンとコマンドブロックで作った脱出マップを3人が攻略する。暗号解読、パズル、トラップ回避。あさめが力技で壊そうとする。まひるが暗号をあっさり解く。ひぐれがトラップに全部引っかかる。
構成：マップ紹介 → Room 1（暗号） → Room 2（パルクール） → Room 3（レッドストーンパズル） → 最終Room（トラップ地獄） → 脱出成功/失敗
伸びる理由：脱出ゲームは「次の部屋はどんな仕掛け？」で視聴維持率が高い。やとらの制作者視点も入れると2倍楽しい。

Y13「マイクラ料理対決 — 一番美味そうな料理を作れ」

チャレンジ建築コメディ10分
マイクラのブロックで「料理」を建築。お題は「ラーメン」「ケーキ」「お弁当」。食べ物をブロックで表現するセンスが問われる。ひぐれの「ラーメン」が丸石の塊。まひるの「お弁当」が蜘蛛の目入り。
構成：お題発表 → 制作（3品×タイムラプス） → 品評会（リアクション芸） → 総合優勝決定
伸びる理由：料理×建築のクロスオーバー。「ブロックでどう食べ物を表現するか」は発想力コンテンツ。SNSで完成品がシェアされやすい。

Y14「マイクラ伝言ゲーム建築 — 前の人の建築を見て次を作れ」

チャレンジ建築コメディ12分
お題「犬」→やとらが建築→あさめはやとらの建築だけ見て何か当てて建てる→まひる→ひぐれ。最後のひぐれの建築が原型を留めているか？ お題「犬」が「キノコ」になってたり。
構成：ルール説明 → 1人目建築 → 2人目「これ何？」→建築 → 3人目 → 4人目 → 全公開＆答え合わせ → 2戦目
伸びる理由：伝言ゲームの崩壊っぷりが面白い。各段階で「元のお題から離れていく」のがビジュアルで見える。

ストーリー・シリーズ系 6本

キャラの関係性を描く。エモと物語で「次も見たい」を作る。

Y15「よふかしくらぶ結成秘話 — 4人はこうして集まった」

ストーリーエモ12-15分
4人が出会う前の話。やとらがサーバーを立てた理由。あさめが「夜中に話せる人がほしかった」。まひるが「リアルの居場所がなかった」。ひぐれが「たまたま」入った。それぞれの孤独から始まって、4人が揃った夜。
構成：やとら視点「ひとりのサーバー」 → あさめ合流「うるさいのが来た」 → まひる合流「静かなのが来た」 → ひぐれ合流「普通のが来た」 → 初めて4人で朝まで遊んだ夜
伸びる理由：オリジン・ストーリーは最強のファン定着装置。各キャラの背景が見えることで全ての掛け合いが深くなる。

Y16「やとらがログインしない日」

ストーリーエモ10分
いつもいるやとらが1週間ログインしない。3人で探しに行く（リアルで）——わけにもいかず、マイクラ内でやとらの拠点を守る。まひるが「やとらくんのレッドストーン壊したくない」。ひぐれが「アイツなしでも大丈夫だろ……大丈夫じゃないわ」。最後にやとらが何事もなく戻ってくる。理由は「PCが壊れた」。
構成：Day 1「あれ、やとらいない」 → Day 3「まだいない」 → 3人で日常を回す → うまくいかない → Day 7 やとら帰還 → 「何してたの」「PC壊れた」 → 安堵
伸びる理由：「いつもいる人がいない」の不安は全視聴者が共感。やとらの存在の大きさを逆説的に描く。オチが日常的なのがリアル。

Y17「マイクラRPG — やとらが作った物語を3人が冒険する」

ストーリーマイクラ20分
やとらがコマンドブロックとNPCで作ったオリジナルRPGマップ。3人がプレイヤーとなって「闇の王を倒す」ストーリーを進める。やとら＝ゲームマスター（裏でリアルタイムにイベント発動）。ストーリーは厨二だがクオリティが高い。
構成：OP映像 → キャラ選択（戦士/魔法使い/盗賊） → 村での情報収集 → ダンジョン → ボス戦 → ストーリーのどんでん返し → エンディング
伸びる理由：マイクラ内RPGは映像として新鮮。やとらの裏方が見えるのがメタ的に面白い。シリーズ化可能（Chapter 2、3…）。

Y18「まひるの秘密のワールド」

ストーリーエモミステリー12分
まひるが「誰にも見せてない」ソロワールドを初公開。中はまひるの内面世界。地下に作られた巨大な美術館（額縁にスクショ）。空に浮かぶ島（「ここで泣いた」と書いてある看板）。最奥部に4人の像がある。「よふかしくらぶに入る前から、ここに4人分の場所を作ってた。……なんでだろ。会う前から居場所を用意してたんだね。あはは、気持ち悪いよね」
構成：「見せたいものがある」 → ワールドに入る → 各部屋を巡る → 3人のリアクション（言葉を失う） → 最奥部の4人の像 → まひるの独白 → 3人の返事
伸びる理由：まひるの内面を建築で表現するのがマイクラならでは。「建築は心の鏡」。コメント欄が感想文で埋まる。

Y19「ひぐれのリアル友達が来た日」

ストーリーコメディ12分
ひぐれが学校の友達をサーバーに連れてくる。リアル友達は「普通の陽キャ」。よふかしくらぶの異常さ（まひるの家、やとらの厨二、あさめのカオス）にドン引きする。ひぐれが「いつもこうなんだわ」と紹介する側に回る。最後にリアル友達が「……また来ていい？」。
構成：友達紹介 → サーバーツアー（ドン引き連発） → 一緒にプレイ（カルチャーショック） → よふかしくらぶの良さを知る → 「また来ていい？」 → ひぐれの「だろ？」
伸びる理由：「外部の人の目」を通すことで、よふかしくらぶの異常さと良さを再発見させる。新キャラ登場で新鮮味。ゲストコラボの布石。

Y20「最後の夜 — サーバーリセット前日」

ストーリーエモ15分
ワールドをリセットすることになった。最後の夜、4人でワールドを歩き回る。思い出の場所、最初のベッド、初めてエンダードラゴンを倒した場所。最後にスポーン地点でスクショを撮って「せーの」でリセットボタンを押す。新しいワールドの最初の画面で終わる。
構成：「明日リセットする」宣言 → 最後の夜の散歩 → 名場面の場所巡り → 各自の最後の作業 → スポーン地点集合 → スクショ → リセット → 新しい朝
伸びる理由：マイクラプレイヤー全員の「ワールドを消す切なさ」に刺さる。新シーズンへの期待も同時に作れる。絶対泣くやつ。

建築・クリエイティブ系 4本

作る楽しさ。タイムラプスが映える。

Y21「4人で巨大よふかしくらぶ拠点を建てる」

建築マイクラ15-20分
共同建築プロジェクト。各自が1エリアを担当。あさめ＝遊び場、まひる＝地下秘密基地、ひぐれ＝広場、やとら＝全体設計＋自動装置。完成まで数日かかる大プロジェクトをダイジェストで。
構成：設計会議 → 各担当決定 → 建築タイムラプス×4人分 → 中間報告（カオス） → 調整 → 完成ツアー
伸びる理由：完成物がサムネイルとして超強い。シリーズの「ホーム」が視覚的に完成することで世界観が固まる。

Y22「マイクラで学校を建てる — 俺たちの理想の学校」

建築コメディ12分
4人がそれぞれ「理想の教室」を作る。あさめの教室は遊具だらけ。まひるの教室は地下で暗い。ひぐれの教室は普通（逆に珍しい）。やとらの教室にはレッドストーンの自動黒板がある。最後に廊下で繋げて1つの学校にする。
構成：「理想の学校」ブレスト → 各自の教室建築 → 見学会 → 廊下で接続 → 完成した学校で授業ごっこ
伸びる理由：「理想の○○」建築は視聴者が「自分ならこうする」とコメントしたくなる。学校テーマはターゲット層にドンピシャ。

Y23「あさめに建築を教えたら天才だった」

建築エモ10分
やとらがあさめにマンツーマンで建築を教える。最初は酷い。でもコツを掴んだ瞬間に「あさめの個性」が爆発。色使いが独創的で、やとらにはない感性の建築が生まれる。やとらが「……負けた」と認める。
構成：「建築教えて！」 → 基礎レッスン（失敗連発） → ブレイクスルーの瞬間 → あさめ独自スタイルの開花 → 完成品にやとら絶句
伸びる理由：成長物語は普遍的に強い。「教える側が学ぶ」のどんでん返し。あさめとやとらの関係性が深まる。

Y24「コメント欄のリクエストを全部建築する」

建築メタコメディ12分
視聴者のコメントから建築リクエストを拾って実際に作る。「東京タワー作って」「まひるの部屋を明るくして」「ひぐれの石の剣を巨大化」。無茶振りにどう対応するかが面白い。
構成：コメント読み上げ → リクエスト選定 → 建築×5-6個 → 完成品披露 → 次回のリクエスト募集
伸びる理由：視聴者参加型は最強のエンゲージメント。「自分のコメントが採用されるかも」でコメント率UP。定期シリーズ化。

ホラー・ミステリー系 4本

マイクラの「夜」の怖さを活かす。よふかしくらぶの「深夜感」にぴったり。

Y25「深夜3時のマイクラ — 洞窟の奥から声がする」

ホラーマイクラ12分
深夜3時。4人が洞窟探検。環境音がいつもと違う。奥から「フゥーン」という音。ただの洞窟音——と思ったら、看板が置いてある。誰も書いてないはずの看板。そこに書いてあるのは——。ホラーを装ったコメディで終わるパターンと、ガチで怖いパターンの2つを用意。
構成：深夜の洞窟へ → 不気味な環境音 → 「何か聞こえない？」 → 看板発見 → 一同恐怖 → 真相（やとらの仕込み or ガチで不明） → 帰り道でもう1つ怖い出来事
伸びる理由：ホラー×マイクラは鉄板。深夜テンション＋洞窟の音＝リアルに怖い。「これ本当なの？」でコメント考察。
